Deine Finanzen,
auf einen Blick

Das Finance Dashboard bündelt all deine Ausgaben und Einnahmen an einem Ort. Rechnungen landen automatisch aus deiner E-Mail im Dashboard, Kontoauszüge kannst du als PDF hochladen — alles übersichtlich nach Kategorien geordnet.

Rechnungen per E-Mail — automatisch

Rechnungen die an manuel.rechnungen@gmail.com weitergeleitet werden, landen täglich automatisch im Dashboard. Kein manuelles Eingreifen nötig.

Damit das funktioniert, muss dein Name beim Rechnungssteller mit dieser E-Mail-Adresse hinterlegt sein.

Kontoauszug importieren — so geht's

  1. 1Im easybank-Konto den Kontoauszug als PDF herunterladen
  2. 2Im Dashboard auf „Importieren" tippen
  3. 3PDF-Datei auswählen
  4. 4Fertig — Buchungen erscheinen automatisch kategorisiert

Alles bereit!

Du bist eingeloggt — damit ist alles erledigt. KI-Kategorisierung, E-Mail-Import und Datensynchronisation laufen vollautomatisch im Hintergrund.

Bei Fragen tippe jederzeit auf das ℹ-Icon oben rechts im Dashboard.
